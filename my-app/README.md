# Personal Knowledge Base Graph

A modern interactive graph editor built with Next.js and React Flow.

---

## ✨ Features

- ➕ Add new nodes dynamically
- ✏️ Edit node title and description
- ❌ Delete nodes and connected edges
- 🔗 Connect nodes with edges (drag & drop)
- 🗑️ Delete edges by clicking
- 💾 Persistent state using localStorage
- 🌙 Dark / Light mode toggle
- 🎯 Clean and intuitive UI

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React Flow
- TypeScript
- CSS (inline styling)

---

## 🧠 Approach

- Initialized graph using seed data (nodes + edges)
- Used ReactFlow for graph rendering and interactions
- Managed state using React hooks
- Implemented persistence using localStorage
- Designed UI for clarity and usability
